About the Studio

✦ Our Mission

What An Adventure Games is an indie game studio crafting 2D and 3D experiences with retro soul and modern edge. We believe games should be immediately playable — no launchers, no stores, just a URL and a browser. Every title is built with web-native technologies and deployed globally on a free CDN.

✦ Pipeline

AI Asset Generation → Sprite Processing → Game Integration → Deployment → Storefront

✦ Tech Stack

HTML5 Canvas JavaScript Web Audio API Three.js WebGL AI Sprites Cloudflare Pages

✦ Philosophy

We're a two-person operation (plus AI collaborators) proving that small teams can ship polished, playable games without massive budgets. Every asset pipeline is automated, every deployment is one push away, and every game is free to play.
