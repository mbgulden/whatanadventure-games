# What An Adventure Games

Indie game studio. 2D and 3D games built with modern web technologies.

**Domain:** whatanadventure.games (on Cloudflare)

## First Title

**[Darius Star: Cyber Coelacanth](https://github.com/mbgulden/darius-star)** — Horizontal retro shoot-'em-up space arcade game. 16-bit cyberpunk aesthetic.

## Studio Pipeline

```
AI Asset Generation (Google Flow Beta / Leonardo.ai)
  → Sprite Processing (Reusable Pillow-based tool)
  → Game Integration (HTML5 Canvas / WebGL)
  → Deployment (Cloudflare Pages)
  → Storefront (whatanadventure.games)
```

## Tech Stack

- **2D Games:** HTML5 Canvas, JavaScript, Web Audio API
- **3D Games:** Three.js / Babylon.js, WebGL
- **Assets:** AI-generated (Imagen 3, Leonardo.ai), post-processed via Python/Pillow
- **Deployment:** Cloudflare Pages (free global CDN)
- **Source Control:** GitHub (mbgulden org)
